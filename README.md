# UE4GameList
收集一些有意思且质量较高的UE4开源游戏项目

## 仿《战神4》 战斧武器相关
作者: Lewis Fiford

主页: https://www.youtube.com/user/popcornplaya206

该项目仿造了《战神4》中武器瞄准，战斧投掷，战斧曲线收回，回手表现，以及击中敌人和破坏物等，纯蓝图项目，很多细节的部分做的很到位，项目很值得一看。
